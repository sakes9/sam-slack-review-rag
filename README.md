# SAM slack review rag

## 概要

Slack経由でRAGによるコードレビューを依頼するアプリケーションです。

コードレビューはリーダブルコードの「**第Ⅰ部 表面上の改善 第2章 名前に情報を詰め込む**」に焦点を当てて行われます。

### デモ動画

`@Reviewer` とメンションすることで、コードレビューを依頼することができます。
